# D-23.1: TUI 主题 (3 preset)

**Sprint**: 1c-revive-2 (2026-06-06)
**范围**: 用户拍板 A + B + D 全要. D-23.1 主题先行, 不动 turn 路径.
**Ship Ritual**: 1 plan + 1 commit + 1 GitHub Release v1.0.7

## 拍板

- **3 preset**: `default` (现状 cyan) / `solarized` (yellow/blue 暖冷对比) / `monochrome` (无前景色, dim+bold)
- **7 role**: header / model / divider / prompt / error / success / toolName
- **选**: env `DEEPWHALE_TUI_THEME` (默认 `default`), 或 CLI `--theme <name>` (在 `TuiModeOptions.theme` 接)
- **invalid**: stderr warning + 退化 `default`, 不抛 (跟 env-gate 风格一致)

## 改动

### tui.ts (1 文件, ~50 行)

1. **新 type + const**:
   - `export type TuiThemeName = 'default' | 'solarized' | 'monochrome'`
   - `export interface TuiTheme` (7 role)
   - `export const THEMES: Record<TuiThemeName, TuiTheme>` (3 preset)

2. **新 helper**:
   - `resolveTuiTheme(themeArg?: string): TuiThemeName` (arg > env > default, invalid 退化)

3. **改 `colorize` 签名**:
   - 旧: `colorize(text: string, color: string): string` (传 ANSI 字符串)
   - 新: `colorize(text: string, role: keyof TuiTheme, theme: TuiTheme = THEMES.default): string` (传 role 查表)

4. **改 12 处 colorize 调用**: 全 file 改 role 传参, 不再传 ANSI.dim 等字符串

5. **改 `horizontalRule` / `formatTuiStatusBar` 签名**: 加 `theme: TuiTheme = THEMES.default` 参数, 不动其它

6. **改 `runTuiMode` 入口**:
   - 加 `const themeName = resolveTuiTheme(options.theme)` 解析
   - 加 `const theme: TuiTheme = THEMES[themeName]`
   - 加 `TuiModeOptions.theme?: TuiThemeName` 字段

### tui-smoke.test.ts (1 文件, +5 test)

1. **resolveTuiTheme 4 sub-test (describe 嵌套)**:
   - 默认 (无 env 无 arg) → default
   - arg 有效 (3 preset) → 返对应
   - arg invalid ('neon') → stderr warning + 退化 default
   - env 命中 + arg 覆盖 env + arg invalid 仍受校验

2. **THEMES 表完整验收**:
   - 3 preset × 7 role = 21 个值非 undefined
   - 3 preset 至少 1 role 不同 (否则 3 个一样)

3. **端到端 theme 接入**:
   - 跑 TUI + `theme: 'monochrome'`, 验横线 ≥ 4 + model 名 + exit 干净
