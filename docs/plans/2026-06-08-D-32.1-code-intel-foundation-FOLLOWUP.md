## API 差异 (跟原 plan 不同)

原 plan 用 native tree-sitter API:
```ts
import Parser from 'tree-sitter';
import TypeScript from 'tree-sitter-typescript';
const parser = new Parser();
parser.setLanguage(TypeScript.typescript);
```

改用 web-tree-sitter (async init):
```ts
import { Parser, Language } from 'web-tree-sitter';
import { readFile } from 'node:fs/promises';
import { join } from 'node:path';

// One-time init (call in test/beforeAll, not module load)
await Parser.init();
const wasmBytes = await readFile(
  join(require.resolve('web-tree-sitter'), '..', 'web-tree-sitter.wasm')
);
const parser = new Parser();
const lang = await Language.load(wasmBytes); // or Language.load(path)
parser.setLanguage(lang);
const tree = parser.parse(source);
```

**`Parser.init()` 必须 await 一次** (loads the runtime wasm). **Grammar 用 `Language.load(wasmPath)`** (Path string OR Uint8Array).

## Pitfalls (opencode 拍板 防翻车, 跟原 plan 类似)

1. **web-tree-sitter `Parser.init()`**: must await BEFORE any `new Parser()`. 放 module top-level (ESM top-level await OK) 或 lazy in test beforeAll.
2. **WASM path resolution**: 用 `createRequire(import.meta.url).resolve` 而非 `import.meta.resolve` (后者 不一定 work in all node versions).
3. **6 grammar packs** 都需 `tree-sitter-*` npm name, 但**不** 装 `tree-sitter` (native main) — pnpm 应该自动 hoist, 但要确认 6 grammar wasm 都在 `code-intel/node_modules/tree-sitter-*/` 下.
4. **pnpm-workspace.yaml**: **不** 需 `allowBuilds` for tree-sitter (WASM 不 build). 删 opencode 留的 `'tree-sitter*': true` 块.
5. **TS strict 守卫**: 4 new tool schema 必须 TS strict pass.
6. **SymbolTree render**: 用 `ink-testing-library` 跟 D-31.4 TUI probe 一致.
7. **Opencode 10-min timeout**: master plan 估 60-90 min, **必 background + notify_on_complete**.
8. **Opencode may not know web-tree-sitter**: 在 plan 头部 强调, 防 它 重新 走 `tree-sitter` native path.
9. **tree-sitter main package 已被 opencode 装**: 需 `pnpm remove tree-sitter` + `pnpm add web-tree-sitter` 替换. **不** 简单加 web-tree-sitter 然后 留 tree-sitter (会 trigger build).
10. **Grammar .wasm 文件 名**: typescript 用 `tree-sitter-typescript.wasm`, bash 用 `tree-sitter-bash.wasm`, 等. 不要用 `grammar.js` (那是 source).
