# D-31.4.7 — youtube_content.getTranscript 修 B-1 P1 bug

## Context
D-31.4 review found P1 bug: `packages/coding-agent/src/tools/youtube-content.ts:53-67` `getTranscript` action
1. Docstring line 5-6 says "getTranscript 走 youtube-transcript npm (留 stub fetcher, 实际 1 tool 装 npm 时引)"
2. Code does NOT use youtube-transcript npm — it does `this.fetcher('https://www.youtube.com/api/timedtext?v=...&lang=en')` directly
