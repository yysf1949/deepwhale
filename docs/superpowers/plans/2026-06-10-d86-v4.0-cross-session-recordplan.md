## Task 1: Write the cross-session recordPlan test

**File:** `packages/coding-agent/test/unit/persisting-task-graph-recorder.test.ts` (append to existing `describe(...)`)

**Test:**

```ts
it('passes recordPlan entries across separate recorder instances pointing at the same file (D-86 v4.0 cross-session plan)', () => {
  // Extends the D-80 cross-session pattern to the recordPlan method.
  // After D-86, all 3 TaskGraphRecorder methods (recordToolCall,
  // recordGoal, recordPlan) are cross-session verified.
  const dir = mkdtempSync(join(tmpdir(), 'dw-taskgraph-reexport-'));
  const file = join(dir, 'taskgraph.jsonl');
  try {
    const a = new PersistingTaskGraphRecorder({ filePath: file });
    await a.recordPlan({
      tasks: [
        { id: 'plan-0', goal: 'first plan from instance A' },
        { id: 'plan-1', goal: 'second plan from instance A' },
      ],
    });

    // New instance loaded from the same file. It must see the plan
    // entries that instance A wrote.
    const b = new PersistingTaskGraphRecorder({ filePath: file });
    expect(b.getPlans().map((p) => p.tasks)).toEqual([
      [{ id: 'plan-0', goal: 'first plan from instance A' }],
      [{ id: 'plan-1', goal: 'second plan from instance A' }],
    ]);
  } finally {
    rmSync(dir, { recursive: true, force: true });
  }
});
```

Run: `./node_modules/.bin/vitest.cmd run packages/coding-agent/test/unit/persisting-task-graph-recorder.test.ts --reporter=verbose`

Expected: this test passes (green-up-front) with the existing PersistingTaskGraphRecorder.
